# betterclauwd

Quick vibecoded site for using claude(and other apis) on browser because bettergpt.chat is no longer being updated.

## Features

- **Multi-Provider Support**: Claude, GPT, Gemini, Grok, DeepSeek
- **100% Local**: All data stored in your browser's localStorage
- **Image Support**: Paste, drag-drop, or click to attach images
- **Markdown Rendering**: Code blocks with syntax highlighting
- **Chat History**: Sidebar with past conversations
- **Settings**: Theme toggle, default model, temperature, system prompts
- **Streaming**: Real-time response streaming from all providers
